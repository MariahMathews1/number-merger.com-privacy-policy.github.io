Privacy Policy for Number Merge

Introduction
This privacy policy governs your use of the software application Number Merge ("Application") for mobile devices that was created by Mathews Apps. The Application is a puzzle game that provides entertainment through number merging gameplay.

The Application collects the following information:

Device Information: We collect device-specific information such as hardware model, operating system version, unique device identifiers, and mobile network information.
Usage Data: We collect information about how you use the app, including game scores, play time, and interaction with ads.
Advertising ID: We collect the Advertising ID provided by your mobile device for advertising purposes.

Information We Do NOT Collect

Personal information such as name, email, or phone number
Precise location data
Contact information from your device
Photos or media files
